# test-ava-jquery

> My AVA jQuery playground

## Install
```
npm i
```

## Build
```
npm run build
```

## Develop
```
npm start
```
Open [localhost:8080/webpack-dev-server/example/](http://localhost:8080/webpack-dev-server/example/)

## Test
```
npm t
```

With watch
```
npm run tdd
```
